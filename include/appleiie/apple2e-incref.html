Apple IIe / 65C02 Quick Reference

Key Memory Locations

  • $0400-$07FF - Text Page 1 (40x24)
  • $0800-$0BFF - Text Page 2
  • $2000-$3FFF - Hi-Res Page 1 (280x192)
  • $4000-$5FFF - Hi-Res Page 2
  • $C000 - Keyboard (KBD)
  • $C010 - Keyboard strobe clear
  • $C030 - Speaker toggle
  • $C050-$C057 - Display mode soft switches
  • $C061-$C063 - Pushbuttons (Open Apple, Closed Apple)
  • $C064-$C067 - Paddle inputs
  • $C070 - Paddle trigger
  • $BF00 - ProDOS MLI entry

Monitor ROM Routines

  • $FC58 HOME - Clear screen, home cursor
  • $FC22 VTAB - Set cursor row
  • $FBC1 BASCALC - Calculate row base address
  • $FD0C RDKEY - Read keypress (waits)
  • $FDED COUT - Print character
  • $FD8E CROUT - Print carriage return
  • $FDDA PRBYTE - Print hex byte
  • $FCA8 WAIT - Delay

65C02 Extra Opcodes vs 6502

  • STZ - Store zero to memory
  • PHX/PHY - Push X/Y to stack
  • PLX/PLY - Pull X/Y from stack
  • BRA - Branch always (relative)
  • TRB/TSB - Test and reset/set bits
  • (zp) - Zero-page indirect addressing
